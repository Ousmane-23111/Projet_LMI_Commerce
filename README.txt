# Projet LMI 2 - Commerce en Ligne
## Ousmane Gueye - Université Iba Der Thiam de Thiès
## Année universitaire 2025-2026

## Description
Système de gestion de commerce en ligne implémentant
5 structures de données en langage C.

## Compilation
1. Ouvrir Dev-C++
2. Ouvrir le fichier Projet_LMI.dev
3. Appuyer sur F9 pour compiler
4. Appuyer sur F10 pour exécuter

## Organisation des fichiers
- structures.h : définition des structs
- tableau_statistique.c/.h : tableau statique
- tableau_dynamique.c/.h : tableau dynamique
- liste_chainee.c/.h : liste doublement chainée
- abr.c/.h : arbre binaire de recherche
- tas.c/.h : tas binaire
- tri.c/.h : algorithmes de tri
- agregations.c/.h : statistiques
- persistance.c/.h : sauvegarde/chargement
- benchmark.c/.h : mesures de performance
- courbes.py : génération des courbes
